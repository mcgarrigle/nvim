# nvim

Linux install nvim

```
git clone https://github.com/mcgarrigle/nvim.git $HOME/.config/nvim
```
then ...
```
./setup.sh

# add to .bash_profile
export PATH="${HOME}/.local/share/nvim/bin":$PATH

```

Export:
```
tar cvzf nvim-complete-linux.tar.gz .local/share/nvim .local/share/nvim-linux64 .config/nvim/
```
